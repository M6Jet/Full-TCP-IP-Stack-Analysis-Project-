🖥️ Full TCP/IP Stack Analysis Project

## 🚀 Overview
hands-on cybersecurity project, focused on a **full TCP/IP stack analysis** using **Wireshark**. The project involved capturing and analyzing network traffic to understand how data moves across each layer of the TCP/IP model—from application to physical layer.

## 🎯 Objectives
- Capture real network traffic and analyze packet flow.
- Understand the interactions between protocols like **TCP, UDP, ICMP, and HTTP**.
- Identify anomalies and troubleshoot network communication issues.
- Apply theoretical networking concepts in a practical scenario.

## 🛠️ Tools Used
- **Wireshark** – For capturing and analyzing packets.
- **Color coding & filters** – To efficiently identify relevant network traffic.
- **PCAP files** – For storing and reviewing captured network data.

## 📝 Key Learnings
- **Layered Network Insight:** Observed data movement across the TCP/IP stack.
- **Protocol Analysis:** Learned to interpret packet headers, flags, and payloads.
- **Practical Cybersecurity Skills:** Monitored network traffic and spotted potential vulnerabilities.
- **Analytical Thinking:** Developed step-by-step troubleshooting skills for complex network interactions.

## 🔜 Next Steps
- Explore **advanced traffic analysis**, including encrypted traffic and anomaly detection.
- Integrate **automated scripts** for packet inspection and analysis.
- Apply these skills in **real-world cybersecurity scenarios** and larger network environments.
